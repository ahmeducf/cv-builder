# CV Builder

A simple CV builder using React.

## Technologies Used

- Vite: build tool chain
- React
- Vanilla CSS
- ESLint: linting
- Prettier: code formatting
- commitlint: commit message linting
- Husky & lint-staged: git commit hooks
